Privacy Policy for Network Debugger Plus

Last updated:

Overview

Network Debugger Plus is a Chrome DevTools extension designed to help developers debug and inspect network requests. This privacy policy explains how we handle your data.

Data Collection

Network Debugger Plus does NOT collect, store, or transmit any user data.

  • No personal information is collected
  • No browsing history is stored
  • No network request data is sent to external servers
  • No analytics or tracking is performed
  • No user accounts or profiles are created

Data Usage

All network request data is processed locally in your browser. The extension operates entirely offline and does not communicate with any external servers except when you explicitly double-click a request to open it in a new tab.

Local Storage

The extension uses Chrome's local storage to remember your preferences:

  • Filter text input
  • Active tab preference (Headers, Payload, or Response)
  • Panel width settings
  • Column width preferences
  • Dark mode preference
  • Form data highlight checkbox states
  • Form data key visibility preferences (which keys to show/hide in form data tables)

This data is stored locally on your device and is never transmitted externally.

Required Permissions

Network Debugger Plus requires the following permissions to function:

  • debugger: To capture network events via Chrome's Debugger API. This is required to monitor network requests in real-time. The debugger API allows the extension to attach to browser tabs and capture network events without needing host permissions.

Note: The extension can open URLs in new tabs when you double-click on a request. This functionality works in DevTools context without requiring additional permissions.

Third-Party Services

Network Debugger Plus does not use any third-party services, analytics, or tracking tools.

Data Security

Since no data is collected or transmitted, there is no risk of data breaches or unauthorized access to your information.

Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected in this document with an updated "Last updated" date.

Your Rights

You have complete control over your data:

  • You can disable or uninstall the extension at any time
  • All local storage data is cleared when you uninstall the extension
  • You can clear extension data manually through Chrome's settings

Contact

If you have any questions or concerns about this privacy policy or how Network Debugger Plus handles your data, please contact us through the Chrome Web Store listing or file an issue on the extension's support page.
